# Мои первые проекты на Python 🐍

Добро пожаловать в мой репозиторий! Здесь собраны мини-проекты, которые я написал во время прохождения курса для начинающих.

## Список проектов:
* **Генератор паролей** (`pass_gen.py`) — безопасный генератор с защитой от некорректного ввода.
* **Шифр Цезаря** (`caesar_cipher.py`) — программа для шифрования и дешифрования текста.
* **Виселица** (`hangman.py`) — консольная игра "Угадай слово" с бесконечным перезапуском.

*Все проекты написаны в VS Code и протестированы в терминале.*
